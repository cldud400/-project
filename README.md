# capstan-design-project
> 포토센서를 이용한 영양제 자동 배출기능을 갖춘 스마트 알림이
- 버튼을 눌러 알람 시간을 설정 한 뒤 realtime과 시간이 같게 되었을 때   배출 버튼을 누르면 영양제가 자동 배출된다.
- 배출버튼은 스마트폰 어플로 연동이 가능하다.
- 서로 다른 영양제 두 가지를 배출할 수 있다.
- 외부 lamp로 잔량 확인이 가능하다.
